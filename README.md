# Proyecto POO en JavaScript con Node.js

Este proyecto es una demostración sencilla de los conceptos fundamentales de la Programación Orientada a Objetos (POO) utilizando JavaScript y ejecutado en un entorno de Node.js.

---

## Objetivo

Aplicar y comprender los principios básicos de la POO mediante un ejemplo práctico enfocado en:

- Herencia  
- Polimorfismo  

---

## Conceptos Aplicados

### Herencia
Se utiliza una clase base (Animal) que contiene atributos y métodos comunes.  
Las clases hijas (Perro, Gato, etc.) heredan estas características y pueden especializar su comportamiento.

### Polimorfismo
Se redefine el método hacerSonido() en cada clase hija, permitiendo que diferentes objetos respondan de manera distinta al mismo método.
